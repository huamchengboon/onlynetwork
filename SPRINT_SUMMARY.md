## Design Language Compliance Checklist

Every sprint must verify:
- [ ] Colors: Only Canvas White, Blueprint Blue, Ink Black, Grid Gray
- [ ] Typography: Pixelated headings, Serif body, Monospace labels
- [ ] Visual Style: Isometric diagrams, blueprint aesthetics
- [ ] UI Components: Sharp corners, blueprint blue accents
- [ ] Layout: Asymmetric two-column where applicable
- [ ] Annotations: FIG_XXX labels, technical legends
